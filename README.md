# Calculadora de Gorjeta

App desenvolvido a fins de estudo com o React Native.


## Executar o App

1. Rodar o comando dentro do projeto:
    ```
    $ npm install
    ```
2. Executar o comando de build do app
    ```
    $ react-native run-ios

    // para Android
    $ react-native run-android
    ```
